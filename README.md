# ⚽ Football Match Management & Ticket Booking System

## 📌 Giới thiệu

Đây là **ứng dụng Desktop mô phỏng quá trình đặt vé và quản lý các trận đấu bóng đá**, được xây dựng trong khuôn khổ **đồ án môn Cơ sở dữ liệu**.

Ứng dụng cho phép quản lý thông tin các trận đấu, đội bóng, sân vận động và mô phỏng quy trình đặt vé của người dùng.  
Hệ thống được phát triển bằng **Java** kết hợp với **Microsoft SQL Server**.

---

## 🖥️ Loại ứng dụng

- Desktop Application
- Ngôn ngữ lập trình: **Java**
- IDE: **NetBeans**

---

## 🛠️ Công nghệ sử dụng

- **JDK**: 21
- **Database**: Microsoft SQL Server
- **JDBC Driver**: 12.10.1.jre11
- **Thư viện hỗ trợ**: JCalendar
- **Cổng kết nối SQL Server**: 1433

---

## ⚙️ Cấu hình cơ sở dữ liệu

- **Server**: SQL Server
- **Username**: `sa`
- **Password**: tùy chọn
- **Port**: `1433`

> ⚠️ Lưu ý: Hãy đảm bảo SQL Server đang chạy và đã bật TCP/IP trong SQL Server Configuration Manager.

---

## 📂 Chức năng chính

- Quản lý thông tin các trận đấu bóng đá
- Quản lý đội bóng, sân vận động
- Mô phỏng quá trình đặt vé
- Lưu trữ và truy vấn dữ liệu bằng SQL Server
- Giao diện Desktop thân thiện với người dùng

---

## 🚀 Hướng dẫn chạy chương trình

1. Cài đặt **JDK 21**
2. Cài đặt **SQL Server** và cấu hình port `1433`
3. Import database vào SQL Server
4. Mở project bằng **NetBeans**
5. Kiểm tra cấu hình kết nối JDBC
6. Run project
